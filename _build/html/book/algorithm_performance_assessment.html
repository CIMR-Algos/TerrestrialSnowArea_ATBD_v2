Algorithm Performance Metrics (MPEF)#

The performance of the TSA algorithm is evaluated by means of classification accuracy. For this, TSA estimates are divided into true positive (TP), false positive (FP), true negative (TN) and false negative (FN) observations. Those observation classes for a single day are used to calculate the daily classification accuracy:

(2)#\[\text{accuracy} = \frac{\text{TP}+\text{TN}}{\text{TP}+\text{FP}+\text{TN}+\text{FN}},\]

which is obtained for all days and subsequently used to derive monthly and total mean accuracy values. In addition, observation classes are binned with respect to corresponding in situ SD which clarifies the ability of the TSA algorithm to detect dry snow for varying snow depth.
